 

Http-Server - Installation  
 
 and run from command line in current directory to install http-server globally

  npm install http-server -g

npm install http-server -save-dev
 
   
 Starting http-server locally

  node bin/http-server

 It statrts the http-server on port 8080 if 8080 is available or otherwise increnents to next value and starts it i.e 8081

  http://localhost:8080/Basic

  Now you can visit http://localhost:8080/React-Render/testPage.html to view the server in browser
